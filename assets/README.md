# Pacelab assets

## App icon
- **icon.png**: 1024×1024 px. Simple "P" lettermark on white background, electric blue (#0A84FF). Clean, minimal, recognizable at small sizes.
- **adaptive-icon.png** (Android): Same or transparent PNG, 1024×1024.

Generate required sizes with Expo:
```bash
npx expo-image-utils icon ./assets/icon.png
```

## Splash screen
- **splash.png**: White background (#FFFFFF), "Pacelab" wordmark centered in black, tagline below: "Train smarter". Recommended 1284×2778 or similar for full-screen.

## Favicon (web)
- **favicon.png**: 48×48 or 96×96 for web.
